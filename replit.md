# PHP Hello World

## Overview
A simple PHP web application that displays "Hello, World" using Bootstrap 5 RTL styling.

## Project Structure
- `index.php` - Main entry point, renders HTML with PHP echo statement
- `LICENCE` - Project license file

## Technology Stack
- PHP 8.4
- Bootstrap 5.3.8 (RTL version via CDN)

## Running the Application
The app runs using PHP's built-in development server:
```
php -S 0.0.0.0:5000
```
